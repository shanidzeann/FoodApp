# FoodApp

- UIKit
- Архитектура - MVP
- Верстка кодом (SnapKit)
- SQLite.swift - для хранения заказа в корзине
- Firebase Authentication
- Заказы хранятся в Firestore
- Kingfisher
- URLSession
- DispatchGroup
- API - spoonacular
- MapKit 
- CoreLocation
- UIPanGestureRecognizer, UITapGestureRecognizer 
- UIViewPropertyAnimator - для анимации сворачивания/разворачивания меню
